# TimeLine Game

TimeLine Game(以下タイムラインゲーム)とは，与えられたカードに書かれた歴史上の出来事を，時系列順に並べるゲームです．

https://timeline-game-dayo.vercel.app
で公開中

## 動機

タイムラインゲームはマイナーよりで，面白いゲームにも関わらず，オンラインで遊べるものが全然なく，ないなら作ってしまおうということで作成しました．
